# Haystack

[deepset Haystack](https://haystack.deepset.ai) `OpenAIChatGenerator` and `OpenAITextEmbedder` accept any OpenAI-compatible base URL. Ferro Labs AI Gateway is a drop-in.

## Install

```bash
pip install haystack-ai
```

## Configure

```python
import os
from haystack.components.generators.chat import OpenAIChatGenerator
from haystack.dataclasses import ChatMessage
from haystack.utils import Secret

generator = OpenAIChatGenerator(
    model="claude-3-5-sonnet-20241022",       # routed to Anthropic by Ferro
    api_key=Secret.from_env_var("FERRO_API_KEY"),
    api_base_url=os.environ["FERRO_BASE_URL"] + "/v1",
)

result = generator.run(messages=[ChatMessage.from_user("Hello from Haystack")])
print(result["replies"][0].content)
```

For embeddings:

```python
from haystack.components.embedders import OpenAITextEmbedder

embedder = OpenAITextEmbedder(
    model="text-embedding-3-small",
    api_key=Secret.from_env_var("FERRO_API_KEY"),
    api_base_url=os.environ["FERRO_BASE_URL"] + "/v1",
)
```

## Verify

```bash
curl http://localhost:8080/v1/chat/completions \
  -H "Authorization: Bearer $FERRO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"hi"}]}'
```
